    org-mode 标题基本格式¶

    STARS KEYWORD PRIORITY TITLE TAGS

    Note

    1. STARS 星号(*) 表示标题级数
    2. KEYWORD TODO 关键词
    3. PRIORITY TODO 任务优先级
    4. TITLE 真正的标题内容
    5. TAGS TODO 标签

    TODO 关键词¶

    关键词分 类型关键词 和 状态关键词 两类

    定义关键词¶

    1. 在 org 文件中定义关键词

    2. 在 emacs 配置文件中定义关键词

      (setq org-todo-keywords
    `((sequence "DOING(p!)" "TODO(t!)" "REVIEW(r!)"
       "|" "DONE(d!)" "ARCHIVE(a@/!)" "CANCELED(c@/!)")
))

      Note

      1. DOING 状态：正在进行
      2. TODO 状态：待完成
      3. REVIEW 状态：复习
      4. DONE 状态：已完成
      5. ARCHIVE 状态：因故暂存
      6. CANCELED 状态：因故终止
      7. "|" 分隔符，区分已完成、未完成状态
      8. (_key_) 使用 C-c C-t 切换状态时的关键词
      9. (!) 设置到改变为该状态时，打一个 时间戳
      10. (@/) 切换到该状态时，需要输入说明文字（如解释项目终止的原因）

    切换关键词¶

    方法一：C-c C-t 选择关键词

    方法二：直接输入

    方法三：（不建议）S-left and S-right 切换关键词

    改变关键词外观¶

    在 emacs 配置文件中加入：

    (setq org-todo-keyword-faces
  `(("DOING"    .   (:foreground "LimeGreen"  :weight bold))
    ("TODO"     .   (:foreground "#FC635D"    :weight bold))
    ("REVIEW"   .   (:foreground "orange"     :weight bold))
    ("DONE"     .   (:foreground "DarkGreen"  :weight bold))
    ("ARCHIVE"  .   (:foreground "#4170C3"    :weight bold))
    ("CANCELED" .   (:foreground "DimGray"    :weight bold))
))

    Note

    1. :background 字体背景颜色（最好别用，太花了）
    2. :foreground 字体颜色
    3. :weight 字重

    TODO 优先级¶

    改变优先级外观¶

    在 emacs 配置文件中加入：

    (setq org-priority-faces
  '((?A . (:foreground "#3058FB"))
    (?B . (:foreground "#2E8DFF"))
    (?C . (:foreground "#61A9FF"))
    (?D . (:foreground "#92C4FF"))
    (?E . (:foreground "#C2DEFF"))
))

    Note

    和 TODO 关键词外观一样

    建议选用一个色系深浅不同的颜色，不然和 TODO 关键词花到一块儿去了，费眼睛

    设置任务时间¶

    日历¶

    使用 M-x calendar 打开 Emacs 日历

    使用 C-u M-x calender 按提示跳转到特定年月

    q 退出日历

    时间戳¶

    用 <> 或 [] 表示时间戳，尖括号表示激活时间，加入日程表；方括号表示非激活时间，不加入日程表

    格式：<日期 时间或时间范围 周期或重复指令>

    如 <2020-10-12 10:20am-3:00pm .+1d>

    周期或重复指令：

    符号 意义
    .

    开始时间¶

    用命令 C-c C-s 输入，如

    SCHEDULED: <2013-06-25 周二>

    到达开始时间后，任务将在每天的日程表上显示，直到任务的状态变为 DONE

    也可以延迟计划任务在日程表上显示的时间，如

    SCHEDULED: <2013-06-25 周二 -2>

    Note

    如果要指定一个任务在确切的时间开始，直接输入时间戳就可以了

    截止时间¶

    用命令 C-c C-d 输入，如：

    DEADLINE: <2013-06-26 周三 -5d>

    后面的 -5d 表示在截止时间前5天开始在日程表上提醒，直到任务的状态变为 DONE

    Note

    如果一个任务既有开始时间，又有结束时间，日程表上将显示两个，可以通过 (setq org-agenda-skip-deadline-prewarning-if-scheduled t) 选择显示哪个

    设置任务标签¶

    C-c C-q 输入标签名并回车，标签自动添加到当前文本所属标题的标签列

    TODO 任务属性¶

    可以为 TODO 任务添加属性

    格式：

    :PROPERTIES:
:属性: 内容
:END:

    例如：

    * TODO 采购清单
    :PROPERTIES:
    :地点: 步步高超市
    :消费限额: 3000 rmb
    :END:
